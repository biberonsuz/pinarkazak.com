PINAR KAZAK WORKS ABOUT

Garbage Island is an online game, aiming to inform the visitors about garbage accumulation. After giving the user textual information about the garbage accumulation, the website moves onto the online game, where punctuation, symbolising garbage accumulation can be clicked and removed from the website. The website aims to create and ismulate the atmosphere of an ocean with audio, animation and visuals, and is an experiment in type setting and storytelling on web, utilising Javascript and CSS.
